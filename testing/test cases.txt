№1  Тестирование подбора отеля
    1. Переходим на www.quirkyaccom.com/
    2. Переходим в Search.
    3. Выбираем бюджет.
    4. Выбираем тип отеля.
    5. Выбираем количество человек, которые будут заселяться, и количество номеров.
    6. Нажимаем кнопку "Искать".
    
    Ожидаемый результат: появится список отелей или квартир, подобранные по нашим параметрам.

№2  Тестирование функции "Семейный режим"
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. На появившейся странице со спсиком жилья, нажимаем кнопку "Семейный режим"
        
    Ожидаемый результат: Список жилья по нашим параметрам обновится. Пояявится список, который будет содержать наименьшую цену жилья и скидки для семей.

№3  Тестирование фильтрации по жилью.
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. В верхней части страницы выбираем локацию, например "England, North".
    4. Далее, видим предложение выбрать количество человек, выбираем 3-4.
    
    Ожидаемый результат: список жилья будет состоять только подходящие отели.

№4  Тестирование раздела помощи
    1. На шапке сайта выбираем "Menu".
    2. Выбираем пункт "how to book".

    Ожидаемый результат: появится страница помощи с подробным описанием шагов для оформления заказа.

№5  Тестирование функции "Trevel Ideas"
    1. На шапке сайта выбираем "Menu".
    2. Выбираем пункт "Travel Ideas".
    3. Выбираем любой пункт из предложенных, например, "Ssshh! Its a SECRET!".

    Ожидаемый результат: появится возможность оформить заказ на отель, предложенный на странице.
